Wu Wei / Chaos is a flow-field–based generative system.
Through particles, noise, and temporal accumulation, it explores the tension between action and non-action. Rather than pursuing efficiency or control, the system treats user input as a disturbance.
Each press injects chaos into the field; each moment of release allows the system to gradually return to calm. Visual motion, colour, and text together form a continuously evolving state, inviting the user to explore the balance between intervention and letting go.
